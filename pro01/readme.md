- Language : Java 11
- Backend Framwork : Spring Legacy 5.3.20

- CSS Framwork : Bulma 1.0
- Application Design : ObjectAid

  - UML : draw.io


- UI/UX Design
  - User Scenario : Creately

  - Story Board : Power Point

- Project Test
  - 단위 테스트 : jUnit 4.13
